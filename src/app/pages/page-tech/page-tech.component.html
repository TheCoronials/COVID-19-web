How we built it

We based our developments on the platform Twilio, which provides a simple integration with WhatsApp API and a simple way to create user journeys. Using the Autopilot tasks. To avoid leaving the user trying different options and entering into frustrating “I didn’t understand”, we implemented a menu system using our Backend API (a python flask application) and helping the user following a simple journey and getting the right information in simple few steps.

This website was built with Angular 9.

Challenges we ran into

The navigation through the chatbot journey was very challenging. The goal was to make the content accessible in the easiest way possible. Twilio do have some basic NLP (Natural Language Processing) technology, but need to be trained with a lot of sentences to be very performant and make it natural to access the right information. The menu feature we implement solve that problem by giving the user the control of his journey and not getting lost while getting the information they need.
